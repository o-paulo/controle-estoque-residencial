# 📦 Controle de estoque residencial #

O programa serve para controlar o estoque de sua residencia, para que voce registre seus itens.

## 🚀 Funcionalidades ##

O programa conta com as seguintes funcionalidades:

- CRIAR itens
- LISTAR itens
- SALVAR itens

## 📲 Como executar ##

Para executar o programa é necessário ter um compilador instalado em seu computador. Neste projeto utilizamos o compilador <b>GCC</b>.

### 🛠️ Compilação
Para compilar o arquivo vá no seu terminal e digite <code>gcc main.c -o main.out</code>.
<br>
Se o terminal não exibir nenhuma mensagem de erro, significa que o programa foi compilador com sucesso!
<br>
Agora basta executar o programa digitando o comando <code>./main.out</code>. Ao apertar 'enter' o programa será iniciado e pronto para ser explorado.
